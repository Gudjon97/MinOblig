Git & GitHub

Git er et versjonskontrollsystem som skal gjøre det lettere å finne fram til en spesifikk versjon av en fil. Det betyr at selv om du arbeider på en fil en lang stund og gjør veldig mye, så kan du fortsatt lett finne fram til en tidligere versjon istedet for å måtte slette ting.

GitHub er en platform som lar deg samarbeide med andre på kodeprosjekter, den utfører også versjonskontroll. GitHub er bygget på Git
